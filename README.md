# Buy Me
Buy Me is an application that will enable companies/investors,developers to buy or sell projects i.e softwares,android application,ios applications.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:ci

# test coverage
$ npm run test:cov
```
